# 最推荐你先定的统一文案基调

建议整款游戏统一成这条风格：

**可爱 + 脑洞 + 轻挑衅 + 不冒犯 + 很想分享**

也就是：

* 不要太正经
* 不要太幼稚
* 不要太毒舌
* 要“有点损，但很可爱”
