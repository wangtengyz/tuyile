
# 一、先定义这个游戏的爆款方向

## 游戏定位

**名字方向：**

* 看图猜猜猜
* 脑洞猜图王
* 一眼会认，二眼烧脑
* 萌图猜词
* 猜图小天才
* 谐音脑洞王

## 核心体验

用户先看到一张“标准认知图”：

* 这是毛巾
* 这是狮子
* 这是叶子
* 这是猪
* 这是草莓

然后第二张变体图出现：

* 毛巾变成端热饮的拟人角色
* 狮子变成拿弓箭射鹰的角色
* 叶子出现在锅里被煮
* 猪在织毛衣
* 草莓倒过来

用户根据第二张图猜：

* 职业
* 动作
* 成语/词语
* 谐音词
* 网络梗
* 生活常识词

这类玩法比传统看图猜成语更容易火，因为：

* 图更萌，停留更强
* 不是老套题库
* 用户会因为“差一点猜到”而上头

# 二、建议你把游戏做成 4 大题型，而不是单一题型

## 题型 1：基础认知 → 职业

格式：

* 上图：这是 X
* 下图：这是 __（职业）

例子：

* 毛巾 + 拿热饮 → 可设计为“服务员 / 茶艺师 / 咖啡师 / 保洁员”等
* 狮子 + 表演 → 驯兽师 / 猎人 / 侦察员 / 射手

**特点：**
最适合小朋友、家长、女性用户，传播最广。

---

## 题型 2：基础认知 → 动作

格式：

* 这是 X
* 这是 __（动作）

例子：

* 猪 + 织毛衣 → 编织 / 打毛线 / 织衣服
* 叶子 + 入锅 → 煮 / 熬 / 烹饪
* 草莓倒置 → 倒立 / 翻转 / upside down

**特点：**
理解成本低，题库最好扩。

---

## 题型 3：基础认知 → 词语 / 成语 / 谐音梗

这才是最容易爆火的部分。

例子方向：

* 狮子射鹰 → 可引导成“射雕英雄”
* 叶子下锅 → “叶公好龙”这种不行，图像不直观；要选直观词
* 猪织毛衣 → “织猪侠 / 针线活 / 贤惠猪”这种偏梗图
* 草莓倒过来 → 倒莓 / 霉运 / 倒霉（这个很强，草莓倒过来≈倒莓≈倒霉）

**这个方向最适合短视频评论区互动。**

---

## 题型 4：双层误导题

上图先给用户安全感，下图反转。

例如：

* 上图：这是叶子
* 下图：叶子在锅里
* 用户以为答案是“煮叶子”
* 实际答案可以做成“茶叶”

也就是第二张图其实在指向**另一个更准确的词**。

这种题型特别上头。

# 三、你这类游戏的真正爆点，不是“猜对”，而是“猜错也爽”

所以玩法必须这样设计：

## 单局节奏

1. 显示原图认知
2. 显示变体图
3. 给 4 个选项
4. 3 秒倒计时
5. 猜错给出“你被第一眼骗了”
6. 解析图像逻辑
7. 引导分享下一题

## 爆款互动机制

你一定要加这几个：

### 1. 评论区挑战话术

* 第 3 题 90% 的人猜错
* 看到第 5 题的人都沉默了
* 小学生 10 秒答对，成年人卡住 1 分钟
* 你能通关说明你脑洞真的大

### 2. 分享裂变

* 发给朋友帮你猜
* 你猜对 3 题，看看 ta 能不能全对
* 家长版 / 情侣版 / 打工人版 / 成语版 / 职业版

### 3. 连续答对奖励

* 3 连对：解锁“脑洞新手”
* 5 连对：解锁“图感达人”
* 10 连对：解锁“谐音王者”

### 4. 错题复活

* 看广告得提示
* 分享好友得半个字提示
* 观看完整解析解锁下一关

# 四、内容生产系统：你以后不要一张张想，要做成“题库公式”

## 你的题目生产公式

### 基础公式

**基础物体 + 拟人化/动作变化 + 答案类型 + 文字版式 = 一道题**

例如：

* 毛巾 + 拿热饮 + 职业 + 上下对照
* 狮子 + 拿弓射鹰 + 职业/角色 + 上下对照
* 叶子 + 在锅中被煮 + 词语/动作 + 上下对照
* 猪 + 打毛线 + 动作/职业 + 上下对照
* 草莓 + 倒置 + 谐音词 + 上下对照

## 方案一：最稳的“二联卡片图”总提示词模板

这个适合直接批量生成和统一风格。

### 总模板

```text
请生成一张中文儿童益智猜图卡片，竖版双分栏布局，上下两块内容，整体风格统一、可爱、简洁、适合幼儿识字和益智游戏传播。

【整体画风要求】
超可爱Q版插画，儿童启蒙卡片风，扁平化，柔和配色，线条圆润，干净背景，居中构图，表情萌，画面简洁，信息明确，适合短视频和小游戏素材。不要复杂背景，不要写实，不要杂乱元素，不要过多阴影。

【版式要求】
- 竖版
- 上半部分一张认知图
- 下半部分一张变化图
- 每一部分都要有大号黑色中文文字说明
- 图片主体必须居中，留白充足
- 边框可爱统一，适合儿童卡片
- 中文文字清晰，粗黑体风格
- 不要水印，不要logo，不要额外小字

【上半部分内容】
画一个：{基础物体}
中文文案：这是{基础物体名称}

【下半部分内容】
把同一个主体做成：{变化后的拟人化/动作/场景}
中文文案：这是{答案留空格式}

【附加要求】
- 上下两部分的主体要明显是同一个对象演变而来
- 下半部分要有强烈反差感和趣味性
- 画风统一，适合“看图猜内容”游戏
- 表情要可爱，能引发分享欲
```

## 线 1：儿童启蒙版

* 认物
* 认动作
* 认职业

## 线 2：全民脑洞版

* 谐音梗
* 反转词语
* 看图猜词

## 线 3：热门主题版

* 打工人版
* 情侣版
* 校园版
* 宝妈版
* 职场版
* 网络热梗版

# 十一、直接给你一套“生产级超级提示词”

你可以拿去反复生成。

```text
你现在是“儿童益智猜图卡片设计师”。

请根据我提供的变量，生成一张适合短视频传播和小游戏题库使用的“看图猜内容”双栏插画卡片。

【目标】
上图用于基础认知，下图用于联想猜答案。风格要统一、可爱、有趣、有记忆点，适合爆款传播和批量题库生产。

【固定风格】
- Q版萌系儿童插画
- 扁平化卡通
- 低复杂度背景
- 中心构图
- 色彩明快柔和
- 主体清晰、轮廓明确
- 大留白
- 适合手机竖屏观看
- 中文文字黑色粗体、清晰易读
- 无水印、无logo、无额外文字
- 整体像幼儿园识字卡 + 脑洞猜图卡

【固定版式】
- 竖版
- 上下两栏
- 每栏一个主图
- 上栏下方文字格式：这是{基础物体名称}
- 下栏下方文字格式：这是{留空格式}
- 两栏风格完全统一
- 下图必须让人看出与上图是同一个对象变化而来

【变量输入】
基础物体：{基础物体}
基础物体描述：{基础物体视觉描述}
变化方式：{变化方式}
变化场景：{变化场景}
答案类型：{答案类型}
留空格式：{留空格式}

【额外要求】
- 下图一定要有趣味性、反差感、分享感
- 图像表达必须清楚，不能让人看不懂动作
- 优先适合儿童和大众用户理解
- 适合批量系列化生成
```

# 十二、最后给你一个最实用建议

你不要一开始就追求“答案特别高级”，而要追求：

**3 秒能看懂图，10 秒能卡住人，揭晓后会说“原来如此”。**

这是爆款猜图的关键。

先做这三类最稳：

* 职业版
* 动作版
* 谐音梗版
